# PTSolnsCore

This repository contains the core platform files used by the PTSolns ecosystem and the PTSolns IDE.

It provides:
- Board definitions and platform configuration
- Core build and compilation support
- Hardware variants and pin mappings for PTSolns-compatible boards

PTSolnsCore is open-source and community-driven, forming the foundation for PTSolns board support within the IDE.
